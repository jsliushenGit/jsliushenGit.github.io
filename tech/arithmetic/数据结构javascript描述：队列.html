# 队列是一种遵循“先进先出”原则的有序集合。比如我们平时生活中排队买票，队头的人是第一个进入队伍的人，同时也是第一个买到票的人。 javascript实现

我们使用数组来模拟队列

function Queue() {
  var queue = [];
}

通常一个队列需要有以下方法：

  • enqueue() 向队尾添加一个元素
  • dequeue() 删除并返回队首元素
  • front() 读取队首元素
  • back() 读取队尾元素
  • queueToString() 显示队列内的所有元素
  • empty() 判断队列是否为空
  • size() 获取队列的长度

一个一个实现

# enqueue

向队尾添加元素，可以直接使用数组的push方法，每次添加的元素都在队尾，也就是数组的最后一位。

function Queue(){
	var queue=[];
    this.enqueue=function(value){
    	queue.push(value);
    }
}

# dequeue

删除并返回队首的元素，也就是数组的第一个元素，可以直接使用数组中的shift方法来模拟实现。

function Queue(){
	var queue=[];
    this.dequeue=function(){
    	return queue.shift();
    }
}

# front

获取队首的元素，也就是数组的第一个元素，可以使用数组下标直接获取，第一个元素的下标是0.

function Queue(){
	var queue=[];
    this.front=function(){
    	return queue[0];
    }
}

# back

获取队尾的元素，也就是数组的最后一个元素，可以使用数组下标直接获取，最后一个元素的下标是数组长度减1.

function Queue(){
	var queue=[];
    this.back=function(){
    	return queue[queue.length-1];
    }
}

# queueToString

以字符串显示队列中所有内容,这个可以直接打印数组，字符串形式可以使用toString()方法来转化

function Queue(){
	var queue=[];
    this.queueToString=function(){
    	console.log(queue.toString())
    }
}

# empty

检查队列是否为空，可以用判断数组的长度是否为0来模拟

function Queue(){
	var queue=[];
    this.empty=function(){
    	return queue.length===0
    }
}

# empty

获取队列的长度，可以使用数组的length属性

function Queue(){
	var queue=[];
    this.size=function(){
    	return queue.length
    }
}

# 测试一下

	var myqueue = new Queue();

    myqueue.enqueue(2);
    console.log(myqueue.dequeue());   //1,删除并返回队首的元素
    myqueue.queueToString();  //2， 以字符串的形式打印队列

# 实际应用

约瑟夫环问题：n个人围城一桌（首位相连），约定从1报数，报到数为k的人出局，然后下一位又从1开始报，以此类推。最后留下的人获胜。

	function Joseph(nameList, num) {
  		var queue = new Queue();
  		for (var i = 0; i < nameList.length; i++) {
    		queue.enqueue(nameList[i]);
  		}

  		var eliminated = '';

  		while (queue.size() > 1) {
    		for (var i = 0; i < num; i++) {
      			queue.enqueue(queue.dequeue());
    		}

    		eliminated = queue.dequeue();
    		console.log(eliminated + " Get out!")
  		}

  		return queue.dequeue()
	}
