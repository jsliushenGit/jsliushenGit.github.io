# 栈是一种遵循“后进先出”原则的有序集合。大家可以用羽毛球筒类比一下，假设一个羽毛球筒里按顺序放了5个羽毛球，编号为1，2，3，4，5,1号羽毛球在最底下，5号羽毛球在最上面，很明显1号球是最先放进筒中的。现在我想拿出4号球，那么我只能先拿出5号球，再才能够拿到4号球，这就所谓的后进先出。5号球是最后进入球筒里的，同样，也是最先被拿出来的。 javascript实现

我们使用数组来模拟栈

function Stack(){
	var stack=[];
}

通常一个栈需要有以下方法：

  • push(): 添加元素到栈顶
  • pop(): 移除并返回栈顶元素
  • peek(): 返回栈顶元素
  • isEmpty: 检查栈是否为空，为空则返回true
  • clear: 移除栈中所有元素
  • size: 返回栈中元素个数。
  • print: 以字符串显示栈中所有内容

一个一个实现

# push

向栈顶添加元素，可以直接使用数组的push方法，每次添加的元素都在栈顶，也就是数组的最后一位。

function Stack(){
	var stack=[];
    this.push=function(value){
    	stack.push(value);
    }
}

# pop

删除并返回栈顶的元素，也就是数组的最后一位元素，可以直接使用数组中的pop方法来模拟实现。

function Stack(){
	var stack=[];
    this.pop=function(){
    	return stack.pop();
    }
}

# peek

返回栈顶的元素，也就是数组的最后一位元素，可以使用数组下标直接获取，最后一位元素的下标是数组长度减1.

function Stack(){
	var stack=[];
    this.peek=function(){
    	return stack[stack.length-1];
    }
}

# isEmpty

检查栈是否为空，可以用判断数组的长度是否为0来模拟

function Stack(){
	var stack=[];
    this.isEmpty=function(){
    	return stack.length===0
    }
}

# clear

移除栈中的所有元素，模拟实现就是清空数组，可以直接将数组的长度赋值为0

function Stack(){
	var stack=[];
    this.clear=function(){
    	stack.length=0
    }
}

# size

返回栈中元素的个数，也就是数组长度，可以使用数组的length属性模拟

function Stack(){
	var stack=[];
    this.size=function(){
    	return stack.length
    }
}

# print

以字符串显示栈中所有内容,这个可以直接打印数组，字符串形式可以使用toString()方法来转化

function Stack(){
	var stack=[];
    this.print=function(){
    	console.log(stack.toString())
    }
}
