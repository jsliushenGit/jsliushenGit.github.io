# 开发环境

前端开发环境的搭建基本就是webpack那一套，各框架也都推出了自己的脚手架，比如vue的@vue/cli。react官方推荐的脚手架是react-create-app,跟着文档玩起来。

# 安装脚手架

npm i react-create-app -g

# 初始化react程序

react-create-app react-app

生成的文档目录结构如下

直观上这个目录比@vue/cli生成的vue程序项目结构更简洁，毕竟react脚手架不友好早有耳闻。
安装依赖跑起来

npm i

npm run start
